OSWAP - Online Store With Administration Panel

This online store is a final Project for SDA's 'Java from Scratch' course. Main aim was to practice using Spring Boot, but we also did some front-end development to get more familiar with front-end technologies. The aim of this application is to sell different products in online store. - Products are divided in categories. Customers can search products, add to cart, proceed to process orders. - Administrators can manage (add, update, delete) products and categories, manage carts and orders.
